F13-lab evaluation 3

part1(teachers' interface):

the teachers interface has the facility of selecting and confirming an assignment or a test through the calendar widget provided below. this calendar only accepts and confirms the task if the subjects' period is present that day. it also takes only task per day for students' convenience. its pops a messagebox up informing that a task is already present for that day. this entered information is stored in a csv file called 'test_schedule'. this csv is accessed again in students' interface.

part2(students' interface):

the students interface is a task manager that records logs of your activities and records future assignments or important tasks through an entry box and stores it in a file. this entered logs show up on the listbox for ease of viewing. a selected date on the calendar also displays that day's task. the csv file from teachers' interface is accessed and added to the listbox as assignments.
